## 🚀 Projects Overview

### 1. ConsoleChats
A terminal-based chat application designed to explore low-level networking and real-time communication.

- Socket programming fundamentals  
- Client-server architecture  
- Lightweight terminal interface

### 2. TermChat
An improved version of ConsoleChats with better structure and scalability.

- Modular code design  
- Enhanced communication handling  
- Focus on clean architecture  

---

### 3. DockTrack
A project focused on tracking and managing structured data efficiently.

- Backend-oriented design  
- Data organization and processing  
- Scalable logic implementation  

---

### 4. Docs
A documentation-focused repository containing structured notes and explanations.

- Technical documentation  
- Learning notes  
- Concept breakdowns  

---

### 5. StreamSocketDemo
A demo project showcasing streaming data over sockets.

- Real-time data transfer  
- Understanding streaming vs request-response  
- Networking experimentation

A custom terminal-based text editor built from scratch.

- Low-level terminal control

Includes detailed notes on the development process.
